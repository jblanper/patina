# Coin Research Resources

A curated list of authoritative sources for researching numismatic features and collector needs. Load this file during Phase II–IV of the scouting workflow.

---

## Catalog Databases

| Source | URL | What to use it for |
|---|---|---|
| Numista | numista.com | The largest community-driven coin catalog; covers ancient to modern; searchable by type, country, era |
| PCGS CoinFacts | pcgs.com/coinfacts | US and world coins; grading data, population reports, auction prices |
| NGC Coin Explorer | ngccoin.com/coin-explorer | Grading standards, variety attribution, census data |
| CoinArchives | coinarchives.com | Auction records for ancient and world coins; used by dealers and museums |
| acsearch.info | acsearch.info | Searchable ancient coin auction archive; strong for RIC/RPC/Crawford references |
| Colnect | colnect.com/coins | Collector community catalog; broad coverage across eras and countries |

## Standards Bodies

| Body | Relevance |
|---|---|
| NGC Grading Scale | 70-point Sheldon scale; standard for US and world coins |
| PCGS Grading Scale | Parallel 70-point scale; most widely used by dealers |
| RIC (Roman Imperial Coinage) | Standard catalog for Roman imperial coins |
| RPC (Roman Provincial Coinage) | Standard catalog for Roman provincial issues |
| Crawford (RRC) | Standard catalog for Roman Republican coins |
| Sear (Greek Coins) | Standard reference for Greek coinage |
