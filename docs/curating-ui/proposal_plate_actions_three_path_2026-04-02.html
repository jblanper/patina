Archetype A

The Icon Trio

Three compact icon+label buttons replace the plain-text row. Each action is immediately scannable by its glyph — the label reinforces meaning without becoming the dominant visual. On narrow secondary slots the labels drop away and a tooltip appears on hover, keeping the strip uncluttered. The destructive Remove inherits the same iconographic language but erupts into deep red on hover — a warning without aggression.

Archetype B

The Overflow Menu

A single discreet button — three dots, the universal sign for "more" — replaces the action row entirely. The caption area stays perfectly clean at all slot widths. Clicking reveals a compact dropdown anchored above the trigger: Replace, Import, then a ruled separator before Remove (colour-coded red). This path treats the plate frame as inviolable, keeping all controls subordinate to the image.

Archetype C

The In-Frame Toolbar

Actions migrate inside the plate frame itself — a slim frosted toolbar slides up from the bottom edge on hover, leaving the caption area entirely empty. The caption reads purely as a label; it carries no interactive burden. On secondary (narrow) slots where hover is unreliable, three ghost-icon buttons sit quietly below the caption at minimal size, always accessible without hover. The toolbar's translucent parchment wash keeps the image visible through the controls.
