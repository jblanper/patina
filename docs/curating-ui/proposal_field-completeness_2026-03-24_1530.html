Archetype A

The Minimalist

All four fields are added at the absolute minimum visual cost. Denomination joins the existing inline subtitle chain without differentiation; year_numeric is fully suppressed; edge_desc gets a plain textarea after Reverse; rarity is a free-text input in the metrics grid. No new vocabulary machinery. The object remains undisturbed.

Archetype B

The Ledger

Maximum archival density. Denomination is elevated as its own labeled row beneath the title — the coin's type designation treated with the same visual weight as its physical metrics. year_numeric is shown in the metrics grid as "Year CE" for cross-referencing. Rarity is a vocabulary autocomplete seeded with the CNR scale (C · S · R · RR · RRR · RRRR). Every field surfaces; nothing is implied.

Archetype C

The Hybrid

The chosen path. Denomination slots into the existing subtitle chain (Path A simplicity) with no visual elevation — it is a peer of mint and year. year_numeric surfaces as a muted monospace parenthetical annotation directly after year_display, enabling error-spotting without a dedicated field. Rarity is vocabulary-backed with the CNR scale (Path B rigour) and lives in the metrics grid after Grade. Edge description is a single textarea in the Scriptorium, with an informative placeholder guiding edge-type vocabulary.
