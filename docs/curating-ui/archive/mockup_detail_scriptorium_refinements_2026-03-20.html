Coin Detail & Scriptorium — Surgical Refinements

Audit Summary

Eleven findings across two pages. Five high-priority corrections.

Both pages demonstrate a coherent archival identity. The refinements below address zone separation failures, a hidden interaction affordance, destructive action ordering, and a form pattern inconsistency in the Scriptorium footer. No structural overhaul is required — these are surgical corrections that materially improve clarity and safety.

CD-C1 · Critical Zoom affordance hidden Plate frame click-to-zoom is CSS-cursor-only. No hover overlay reveals the interaction.
CD-H1 · High Delete precedes Edit Destructive action appears first in the header. Both use identical btn-minimal styling.
CD-H2 · High No metrics zone border Metrics grid transitions to Obverse/Reverse descriptions without a hairline divider.
CD-H3 · High Hardcoded "PLATE V" Plate caption always reads "PLATE V" regardless of coin or image context.
SC-C1 · Critical [AUTO-ISSUE] token visible Meta line shows square-bracket debug tokens when metal field is empty (every new coin).
SC-H1 · High Footer breaks form grammar Inline label/input pattern in footer contradicts the vertical label-above/input-below system used everywhere else.
SC-M2 · Medium No obverse primacy All three plate slots render at equal opacity. Obverse should be primary.
SC-M3 · Medium Subtitle column too narrow "DENOMINATION" at 100px column clips at 1.5px letter-spacing. Needs 130px.
