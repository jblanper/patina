Archetype A // The White Cube

Sensory Sanctuary

Prioritizes the visual impact of the numismatic object. All metadata is secondary to the physical form, revealed only when the curator focuses their attention (hover). This mimics the experience of looking at a coin in a drawer before checking the label.

Athens Owl Tetradrachm
AR // 17.20g // 24mm
Byzantine Solidus
AU // 4.50g // 20mm
CURATORIAL AUDIT:
  • Contrast Ratio: 12.5:1 (Ink on Parchment)
  • Focus Indicator: High-contrast sienna outline on focus-visible.
  • Breathability: Maximum. No visual noise until interaction.
  • Empty State: Fallback to a debossed circular silhouette.
Archetype B // The Archival Index

Technical Precision

A dense, scholarly approach where metrics are immediately available for comparison. Ideal for large collections where the curator is looking for specific technical attributes without needing to interact with every card.

  • Breathability: Archival. Structured but dense.
  • Metric Precision: Strict adherence to the 2-decimal rule.
Archetype C // The Scholarly Reveal

Balanced Narrative

The modern standard for digital archives. It balances the aesthetic prestige of the "White Cube" with the utility of the "Ledger". Primary identification is always visible, while scholarly details emerge on focus.

Athens Owl Tetradrachm
17.20g AR
REF. RIC II 218 // Attica, Athens
Byzantine Solidus
4.50g AU
REF. DOC 12 // Justinian I
CURATORIAL AUDIT:
  • Contrast Ratio: 7:1 for core metrics.
  • Focus Indicator: Scale transition (1.02x) + title underline.
  • Breathability: Balanced. Progressive disclosure prevents overwhelm.
  • UX: Predictable and accessible.
