## Bug Prevention (from Post-Implementation Notes)
- **Avoid display: none or opacity: 0 on file input** - these cause mobile browsers to block the input
- Use visible/styled button approach or a properly positioned label
- Keep the DOMContentLoaded event listener for reliable mobile behavior
