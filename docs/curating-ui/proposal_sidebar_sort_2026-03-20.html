Archetype A

The Minimalist — Inline Sort Bar

A single compressed row that lives just above the Reset button. The current sort field cycles on click; the direction arrow flips independently. The archive speaks in whispers — sort is a utility, not a feature, and the sidebar should not dramatise it.

Curatorial Audit — Path A

Contrast (WCAG)

Pass AA — Ink/Parchment 14.3:1. Accent/Parchment 4.7:1 (meets AA for large text).

Sidebar Footprint

Adds ~36px to sidebar height. No layout impact; existing gap absorbs it cleanly.

Discoverability

Moderate. The cycle affordance is subtle — new users may not recognise the field label as clickable. Mitigated by the underline treatment.

Archival Tone

Excellent. Mono typography throughout. Zero visual noise added to the sidebar.

Interaction Model

Click label → cycles Year → Title → Acquired → Year. Click arrow → flips ↑/↓. Two independent targets.

Focus / Accessibility

Both buttons are keyboard-focusable. Tab order: field cycle → direction → Reset.

Archetype B

The Ledger — Radio Rows + Direction Toggle

A dedicated "Order By" filter-group section that mirrors the Era and Metal blocks precisely — same rhythm, same label anatomy, but with circular radio indicators instead of square checkboxes to distinguish single-select semantics. Direction is surfaced as an explicit two-segment toggle beneath. Maximum legibility; the scholar's preference.

Curatorial Audit — Path B

Contrast (WCAG)

Pass AA — Active dir-btn: white/Sienna 4.7:1. Radio dot: Sienna on Parchment clear at 4.7:1.

Sidebar Footprint

Adds ~130px. Sidebar may require scroll on smaller windows. Most desktop heights handle this without issue.

Discoverability

Excellent. Mirrors existing filter groups exactly — zero new interaction model to learn. Asc/Desc labels are explicit.

Archival Tone

Strong. Radio circles maintain visual harmony with checkboxes while signalling single-select semantics correctly.

Interaction Model

Click row to select field (radio). Click segment to choose direction (toggle). Familiar, explicit, zero ambiguity.

Focus / Accessibility

All rows and buttons keyboard-focusable. Direction toggle segments are individually tab-able with clear active state.

Archetype C

The Hybrid — Chip Row + Rotating Arrow

Three compact field chips sit in a single horizontal row, capped by a single direction arrow that rotates 180° on toggle. This path occupies the smallest possible vertical footprint while keeping all three sort fields simultaneously visible. The ink-filled active chip uses the darkest palette token for maximum legibility — a deliberate departure from the accent colour to reserve Sienna for filter selections.

Curatorial Audit — Path C

Contrast (WCAG)

Pass AA — Active chip: Parchment on Ink 14.3:1. Inactive chip text on Parchment 4.6:1.

Sidebar Footprint

Smallest footprint of the three — adds ~80px. Chips fit in a single row at 280px with comfortable padding.

Discoverability

Good. All three options always visible. Arrow label is explicit (Ascending / Descending). Ink-filled active chip is unambiguous.

Archival Tone

Strong. Ink-on-parchment chip inversion echoes the physical stamp. Arrow rotation is the one animated element — used sparingly and purposefully.

Interaction Model

Click chip to select field (any). Click arrow/label to flip direction. One active chip at a time; always one active.

Focus / Accessibility

All chips and arrow button are keyboard-focusable. Active chip's high contrast satisfies focus-visible requirements without extra outline.
