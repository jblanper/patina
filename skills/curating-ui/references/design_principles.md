# Design Principles (The "Gallery Audit")

To maintain the high standards of the Patina project, all UI elements must adhere to these foundational principles:

1.  **Never Touch the Art:** UI elements must not overlap or crowd the coin imagery. The coin is the focal point; the interface is the frame.
2.  **The Digital Sanctuary:** Adhere strictly to the aesthetic philosophy defined in `docs/style_guide.md`. Use contrast and whitespace to create hierarchy and depth.
3.  **Prestigious Utility:** Interaction must be frictionless and intuitive for a serious, potentially older, demographic. Favor clarity, large touch/click targets, and high contrast over visual trends.
4.  **Archival Tone:** Maintain the somber, respectful tone of a high-end physical gallery in all copy, empty states, and error messages.
5.  **Information Hierarchy:** Display only what is necessary for the current context. Use progressive disclosure to reveal technical numismatic data without overwhelming the visual sanctuary.
6.  **Responsive Integrity:** Ensure the curatorial experience translates seamlessly across all screen sizes as defined by the responsive rules in the primary style guides.
