# Numismatic Technical Metrics

## 1. Weight (Grams)
- **Standard:** Use grams (g) to two decimal places (e.g., 17.20g).
- **Measurement:** Use a digital scale. For high-value coins, weight is a primary diagnostic for authenticity.
- **Tolerances:** Ancient coins have wide weight variances, but they should fall within the standard for their period.
- **Heavy Coin:** Above average weight - can indicate under-struck flan.
- **Light Coin:** Below average weight - can indicate clipping or wear.
- **Conversion:** 1 troy ounce = 31.1035g (use for modern bullion).

## 2. Diameter (Millimeters)
- **Standard:** Use millimeters (mm) to one decimal place (e.g., 19.5mm).
- **Measurement:** Measure at the widest point of the flan using calipers.
- **Measurement Tips:**
  - Digital calipers for accuracy
  - Measure both axis if oval (due to uneven striking)
  - Record both if significantly different

## 3. Die Axis (The Clock Rule)
- **Concept:** The relative orientation of the obverse die to the reverse die.
- **Notation:** Use "O'Clock" positions (12, 1, 2, ..., 11).
- **Measuring:** Hold coin with obverse up, rotate until top of reverse aligns with 12 o'clock position.
- **Common Standards:**
  - **6h (180°):** "Coin alignment" - reverse upside down relative to obverse (common in US coinage).
  - **12h (0°):** "Medal alignment" - reverse right-side up (common in UK and ancient Roman coinage).
  - **Other positions:** 1h through 11h indicate varying degrees of rotation.
- **Recording:** If the reverse is upside down relative to the obverse, it is **6h**.
- **Variance:** Die axis can vary within a single issue; catalog the actual position.

### Fineness
- **Fineness:** The proportion of precious metal in an alloy (e.g., .900 = 90% pure).
- **Milliéme:** Alternative fineness notation (e.g., 900‰ = .900).
- **Examples:**
  - Roman denarius: .800-.950 silver (varies by period)
  - Imperial aureus: .950-.980 gold
  - US silver dollar: .900 silver, .100 copper

## 5. Thickness
- **Measurement:** Use calipers to measure flan thickness.
- **Units:** Millimeters (mm) to one decimal place.
- **Significance:** Can indicate authenticity and mint.

## 6. Edge Types
| Type | Description |
|------|-------------|
| Plain | Smooth, no special treatment |
| Reeded | Vertical grooves (milled edge) |
| Lettered | Words or inscriptions on edge |
| Interrupted | Pattern of reeding and plain sections |
| Scalloped | Wavy edge pattern |

## 7. Strike Quality
| Term | Description |
|------|-------------|
| Full Strike | All details rendered clearly |
| Weak Strike | Areas of design missing or faint |
| Soft Strike | Overall weak detail |
| Adjustment Marks | Evidence of file work to reduce weight |

## 8. Photography Standards
- **Lighting:** Even, diffused light to avoid hotspots.
- **Background:** Neutral gray or white.
- **Orientation:** Obverse at top, reverse at bottom.
- **Scale:** Include ruler or scale in at least one photo.
- **Resolution:** Minimum 300 DPI for print; 72 DPI sufficient for web.
- **Angles:** Include both sides, plus edge detail if notable.

## 9. Storage Metrics
- **Humidity:** 45-55% RH for paper/soft storage; 35-45% for raw coins.
- **Temperature:** 18-21°C (65-70°F) stable.
- **Light:** Minimal UV exposure; store in darkness when not examining.
- **Air Quality:** Low sulfur compounds to prevent toning on silver.
